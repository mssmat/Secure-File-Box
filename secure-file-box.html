使い方

送信側は相手の公開鍵だけを使います。

送るときの流れ
  1. 受信者が「鍵管理」で鍵ペア生成
  2. 受信者は公開鍵だけ送る
  3. 送信者がその公開鍵でファイルを暗号化
  4. できた .sfbx を送る
  5. 受信者は自分の秘密鍵で復号
